# node-console-pong
Pong to be played on a node terminal
It is written in typescript

# Usage 
Execute on a terminal 
```
npm install && npm start
```

# Develop
Install:
```
npm install tsc -g
npm install jasmine -g
npm install watch -g
npm install nodemon -g
```

Run on separate terminals:
```
watch jasmine
nodemon built/app.js
tsc -w
```
